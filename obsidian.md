# Obsidian Cheatsheet

## Keybindings

### Common

| Action                 | Keybind             |
| ---------------------- | ------------------- |
| Open link in Edit Mode | `^ + click on link` |
| Toggle edit/preview    | `^ + E`             |
| Open Quick Switcher    | `^ + O`             |
| Search                 | `^ + ⇧ + F`         |
| Back                   | `^ + ⌥ + ←`         |
| Forward                | `^ + ⌥ + →`         |
| New note               | `^ + N`             |
| Open note in new pane  | `^ + click on note` |
