#### Move Cursor

| Action                  | Windows       | MacOS | Linux |     |
| ----------------------- | ------------- | ----- | ----- | --- |
| Move one word at a time | `^ + ←/→`     |       |       |     |
| Move beginning of line  | `Fn + ← or ↖` |       |       |     |
| Move end of line        | `Fn + → or ↘` |       |       |     |

| Move Page Down          | `Fn + ↓ or ⇟` |       |       |     |
| Scroll Line Up or Down  | `^ + ↑ or ↓`  |       |       |     |
