# OS Keybindings Cheatsheet

## Keybindings

### Practice :hammer:

- Navigation
  - windows
  - tabs
  - files
- Text Editing
  - Document formatting
  - Paragraph
  - Delete word, line
- Screenshoot

## Keys

| Action        | Windows  | MacOS    | Linux |     |
| ------------- | -------- | -------- | ----- | --- |
| Page Up key   | `Fn + ↑` | `Fn + ↑` |       |     |
| Page Down key | `Fn + ↓` | `Fn + ↓` |       |     |
| Home key      | `Fn + ←` | `Fn + ←` |       |     |
| End key       | `Fn + →` | `Fn + →` |       |     |
